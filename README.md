# crypto-tracker


## Installation

1. Clone the repository:
   git clone https://github.com//Vk1608/crypto-tracker.git

## To run backend
1. Open terminal in current directory
2. cd crypto-tracker-backend
3. npm install
4. npm run build
5. npm start

## To run frontend
1. Open terminal in current directory
2. cd crypto-tracker-frontend
3. npm install
4. npm run build
5. npm start
